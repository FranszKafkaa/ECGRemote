<p>Requisitos para rodar na própria máquina </p>
<ul>   
    <li>NodeJS</li>
    <li>Python 3.x</li>
    <li>Biblioteca Python biosppy</li>
</ul>

rode o comando
<br>

```bash
    #rodar no docker 
    docker build -t "<nome da imagem>" ./

    docker run "<nome da imagem>"

    #rodar localmente 
    npm install ou yarn install
    
    pip3 install biosppy 
```
